image

So far we defined, what the image of an element under a function is. Now we define, what the image of a function is.

the image \(f(A)\)
Let \(f : A \rightarrow B , f(x) = y \) be a function. Then the image of \(f\) is the set of all images of the elements \(a \in A\), that is \[ f(A) = \{ f(x) | x \in A \} .\]

graph

The graph of a function is the set of all ordered pairs \((x,y)\), where \(x\) is an element in the domain and \(y\) is the image of \(x\).

graphs
Let \(f : A \rightarrow B , f(x) = y \) be a function. Then the graph of \(f\) is defined as \[ \text{graph}(f) = \{ (x,y) | x \in A, y = f(x) \} .\]

An example is the graph of the function \(f : \mathbb{R} \rightarrow \mathbb{R}, f(x) = x^2 \), which is given by \[ \text{graph}(f) = \{ (x,y) | x \in \mathbb{R}, y = x^2 \} ,\] or just \[ \text{graph}(f) = \{ (x,x^2) | x \in \mathbb{R} \} .\] The graph can be drawn as follows:

6
well defined

We will take closer look on how to define mathematical objects without contradictions and apply this concept to functions.

well defined objects
A mathematical object is well defined, if there are no contradictions implied by the definition of the object.

well defined functions
A function is called well defined, if it indeed fullfills the definition of a function. That is, every element in the domain is mapped to exactly one element in the codomain.

Additionally the rule of assignment can be applied to any element in the domain and it maps every element in the domain into the codomain.
Question 1 point
Is the function \[f: \{ 1,2,3 \} \rightarrow \{ 1,2,3 \}, f(x) = x-1\] well defined?
Yes
No
The function \(f\) is not well defined, since \(f(1) = 0\) and \(0\) is not an element of the codomain.
Question 1 point
Is the function \[f: \mathbb{N} \rightarrow \mathbb{N}_0, f(x) = x-1\] well defined?
Yes
No
The function \(f\) is well defined, since every element in the domain is mapped to exactly one element in the codomain. Furthermore the rule of assignment can be applied to any element in the domain and maps it to an element in the codomain.
Question 1 point
Is the function \[f: \mathbb{R} \rightarrow \mathbb{R}_-, f(x) = x\ \] well defined?
Yes
No
The function \(f\) is not well defined, since the codomain is the set of negative real numbers, but the function maps to the set of all real numbers. Thus \[f(x) = x \notin \mathbb{R}^- \ \ \ \text{for} \ \ \ x >0. \]
Question 1 point
Is the function \[f: \mathbb{R} \rightarrow \mathbb{R}, f(x) = \sqrt{x} \] well defined?
Yes
No
The function \(f\) is not well defined, since roots of negative numbers are not defined. For example \( -1 \in \mathbb{R} \), but \[f(-1) = \sqrt{-1} \notin \mathbb{R} .\] (Note: it is possible to define the square root of negative numbers, with complex numbers, but we will not do that here.)
Question 2 points
Which of the following functions are well defined?
\(f: \mathbb{R} \rightarrow \mathbb{R}, f(x) = x^2 \)
\(g: \mathbb{R} \rightarrow \mathbb{R}, g(x) = \sqrt{x} \)
\(h: \mathbb{R} \rightarrow \mathbb{R}, h(x) = \frac{1}{x} \)
\(i: \mathbb{R}_+ \rightarrow \mathbb{R}_+, i(x) = \frac{1}{x^2} \)
\(f\) is well defined, since the square of any real number exists and is real and unique. \(g\) is not well defined, since roots of negative numbers are not defined. \(h\) is not well defined since \(h(0) = \frac{1}{0} \) is not defined. \(i \) is well defined since \(0 \notin \mathbb{R}_+ \).
