introduction

We will start introducing some basic notions for sets, which will turn out usefull for describing all kind of objects. A set is a collection of objects \(a,b,c, \ldots \) of the form \(\{a,b,c, \ldots\} \). If this set is called \(A\), then we write \(A=\{a,b,c, \ldots\} \) and \(a \in A \) if \(a \) is an element of \(A\) and \(a \notin A\) if not. Let's start with some basic examples.

examples
  • \( \{ 1,2,3,4, \ldots \} \) the set of the first four numbers.
  • \( \{ 2, 4, 6, 8, 10 , \ldots \} \) the set of even numbers.
  • \(\emptyset = \{ \} \) the empty set, it does not contain any elements.

2
subsets

In order to only describe parts of sets, we can take a certain amount of elements out and put it in a new set, which we call a subset of the previous set.

subsets
Let \(A,B\) sets.
  • We call \(B\) a subset of \(A\), if all elements of \(B\) are contained in \(A\).
    Notation: \(B \subseteq A\)
  • \(B\) is not a subset of \(A\), if there is an element in \(B\) which is not in \(A\).
    Notation: \(B \nsubseteq A\)
  • \(A\) and \(B\) are called equal, if \(A \subseteq B \) and \(B \subseteq A \) holds.

Elements of the set \(\{\{1\}, \{2\}\}\) have the form \( \{1\}, \{2\}\). and subsets would need another bracket, like \( \{ \{1\}\}\). Thus answers 1,3 and 4 are wrong.

Question 1 point
If for the sets \(A,B\) and \(C\) holds \(A \subseteq B\) and \(B \subseteq C\), does also \(A \subseteq C\) hold?
Yes
No
The statement can be written as \[(A \subseteq B) \wedge (B \subseteq C) \Rightarrow (A \subseteq C). \] We can rewrite this as \[(a \in A \Rightarrow a \in B ) \wedge (a \in B \Rightarrow a \in C ) \Rightarrow (a \in A \Rightarrow a \in C). \] Since the Implication is transitive, we have that the latter expression is true (remember the table of truth for implication), which means, that \(A \subseteq C\) indeed follows.
Question 1 point

Yes
No
A counterexample is given by \(A = \{1,2,3\} , B = \{1,2,4\} , C = \{1,2,3\}\).

numbersets

Sets are used to describe space in it's most elementary form. Any space, such as the threedimensional space, a time horizon or a 2D-plane can be described with appropriate sets. The following number sets are important building blocks.

numbersets
  • \(\emptyset = \{ \} \) the set which does not contain any elements.
  • \(\mathbb{N} = \{ 1,2,3,4, \ldots \} \) the set of natural numbers.
  • \(\mathbb{N}_0 = \{ 0,1,2,3,4, \ldots \} \) the set of natural numbers including zero.
  • \(\mathbb{Z} = \{ \ldots , -2, -1, 0, 1, \ldots \} \) the set of natural numbers.

implicit and explicit set notation

There are two fundamentally different ways to notate a set \(A\), either implicitly or explicitly. The first notation consists of a random symbol which stands for an element of the set for example \(a\), then a line \(\vert\) followed by a description, such as "\(a\) is equal to \(1\)". The explicit notation simply lists up the elements.

examples
  • the explicitly written set \( \{1\} \) can be written implicitly by \[A = \{ a \ \vert \ a=1 \ \}. \]
  • Sometimes we add a set to the left side, to indicate where the elements come from: \[A = \{ a \in \mathbb{N} \ \vert \ a=1 \ \}. \]

numbersets II

We will introduce some further notations for numbersets \(A \subseteq \mathbb{R} \). A plus sign like \(A^+\) indicates, that we remove all elements which are not positive including zero. The same holds for a minus in the superscript, as \(A^-\).
On the other side we can multiply all elements of \(A\) by another \(x \in \mathbb{R}\) and call this set \(x \cdot A\) or \(xA\). The same holds for adding an element \(y \in \mathbb{R}\) to all elements of \(A\).

set notations
Let \(A \subseteq \mathbb{R} \) and \(x,y \in \mathbb{R} \). Then we define the following notations.
  • \(A^+ = \{ a \in A \ \vert a > 0 \ \} \)
  • \(A^- = \{ a \in A \ \vert a < 0 \ \} \)
  • \(x \cdot A = \{ x \cdot a\ \vert \ a \in A \} \)
  • \(A + y = \{ a +y \ \vert \ a \in A \} \)

the powerset

If we are looking at subsets \(A_1, A_2 , \ldots \) of a given set \(A\), it is common to summarize all of these in a new set \( \{A_1, A_2 , \ldots\} \), let's say, if we want to list up all the information. This set is called the powerset of \(A\).

The power set \(\mathcal{P}(A)\)
Let \(A \) be a set. Then \[\mathcal{P}(A) := \{ B\, \vert \, B \subseteq A \, \}\] is the set of all subsets of \(A \) and we call it the powerset of \(A\).
Question 1 point
Which of the following sets is the powerset of \(\{ 1 \}\)?
\(\{ 1 \}\)
\(\emptyset\)
\( \{ 1, \emptyset \}\)
\( \{ \{1 \}, \emptyset \}\)
It holds \(\{1\} \subset A\) and \(\emptyset \subset A\) so both are elements of the powerset. But there are no more, so the last answer is correct.
Question 1 point
Which of the following sets is the powerset of \(\{a,b,c\}\)?
\( \mathcal{P}(A)= \{\{a\},\{b\},\{c\}\}\)
\( \mathcal{P}(A)= \{\{a\},\{b\},\{c\}, \{a,b\}, \{a,c\}, \{b,c\}, \{a,b,c\}\}\)
\( \mathcal{P}(A)= \{\{a\},\{b\},\{c\}, \{a,b\}, \{a,c\}, \{b,c\}, \{a,b,c\},\emptyset \}\)
\( \mathcal{P}(A)= \{a,b,c, \{a\},\{b\},\{c\}, \{a,b\}, \{a,c\}, \{b,c\}, \{a,b,c\}, \emptyset\}\)
\(a,b,c\) are elements, no subsets so the last answer is wrong. All the listed sets in the third answer are indeed subsets of \(A\) and there are no more.
