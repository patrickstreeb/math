2
metric properties of the distance

We will now show that the distance has some properties that we would expect from a distance. Those are called the metric properties. And any object which fullfills them is called a metric.

metric
Let \(X\) be a set. A metric on \(X\) is a function \(d: X \times X \to \mathbb{R}\) such that for all \(x,y,z \in X\) the following properties hold:
  • \(d(x,y) \geq 0\)
  • \(d(x,y) = 0 \iff x=y\)
  • \(d(x,y) = d(y,x)\)
  • \(d(x,z) \leq d(x,y) + d(y,z)\)

3
epsilon \(\varepsilon \)

We will introduce a key concept in analysis, the epsilon technique. The basic idea is to say instead of saying an object has value zero, it is smaller than any positive, and larger than any negative number. Thus it can only be zero.

\(\varepsilon\)-technique 1
Let \(x \in \mathbb{R}\) and \(\varepsilon > 0\). Then it holds \[ x=0 \iff |x| < \varepsilon \]
proof.
  • Let \(x=0\). Then \(|x| = 0 < \varepsilon\).
  • Let \(|x| < \varepsilon\). Then \(x < \varepsilon\) and \(x> - \varepsilon\). Because it holds for all \(\varepsilon\) that means \(x \geq 0\) and \(x \leq 0\), so \(x=0\).
